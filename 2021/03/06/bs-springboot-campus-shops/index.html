【毕业设计】基于Springboot+layui+vue的校园商铺系统

发表于 2021-03-06 | 分类于 Java | 阅读次数:

导语

需要源码的可以加QQ群：994793967，私信群主

一、综述

二手市场在中国是一个不够完善和成熟的市场，尤其在大学校园内。随着经济的发展，大学生生活水平的不断提高，闲置物品对很多人都是比较麻烦的问题，“二手交易”的需求可以说是越发强烈，大学生手中有大量二手商品闲置 ，但因校园缺乏专业化、标准化的二手交易平台，而让这些二手商品无法充分利用。而且校园二手交易市场目标顾客群已经形成，学生之间有许多共同的兴趣和消费倾向，消费商品较具有趋同性，为二手交易市场提供了条件。与此同时，网络购物的兴起，校内交易的可信性，便利，时效等考虑，校园二手市场有一定的基础。因此，开发一款面向大学校园的二手交易商城是很有必要的，前景十分广阔。

二、可行性分析

1.技术可行性

本系统采用SpringBoot系统的架构，采用Java作为开发语言。使用Vue作为页面渲染，JavaScript进行数据交互，MySQL进行数据存储，实现难度适中，可以进行开发。

2.硬件可行性

随着人们生活水平的，电脑手机逐渐成为了人们必不可少的生活用品，所以对于本课题所需求的硬件设备也十分常容易得到，可行性较高。

3.操作可行性

对于网购流行的当下，线上二手交易商城上手更加容易， 用户可以很容易的浏览网站，寻找自己喜欢的商品。

4.经济可行性

在开发的时候只需要适当的人力物力就可以完成，不浪费不必要的资源。所以前期开发成本较低，后期在使用本软件时会产生验证码费用，费用由对应验证码提供商收费。

三、完成本课题所必须的工作条件

  • 操作系统： Window 10操作系统的笔记本电脑
  • 软件开发工具：IDEA，MySQL，Navicat for MySQL
  • 其他工具：亿图图示进行数据库以及流程分析
  • 运行环境：Java JDK1.8

四、主要功能

登陆、注册、重置密码模块

  1. 登录：登录可以使用手机号或用户名进行登录

  2. 注册的时候会判断账号是否以及存在，用户名是否已存在。

    会判断手机号及邮箱的格式，但不会判断邮箱是否已使用过。

    发送短信验证时也会对手机号进行查重，验证码五分钟内有效。

  3. 重置密码时会验证手机号格式，判断是否存在该用户， 短信验证码五分钟有效。

私信聊天模块

  1. 用户对于感兴趣的商品可在详情页面点击感兴趣私信商品发布者
  2. 用户聊天页面展示对方是否在线的状态
  3. 记录用户之间的聊天记录

商品模块

  1. 发布商品
  2. 修改商品
  3. 商品详情界面
  4. 首页界面
  5. 商品列表界面
  6. 商品搜索界面

评论模块

  1. 查看商品下所有评论及回复
  2. 评论回复以及对应删除操作

个人中心

  1. 修改个人信息
  2. 修改密码
  3. 分页展示个人各类商品信息 已审核:1，待审核:3，违规:0，已完成:4
  4. 个人对商品的操作 删除:2 已完成:4
  5. 更换手机号

收藏模块

  1. 商品详情界面：收藏商品or取消收藏
  2. 收藏列表界面取消收藏
  3. 分页查看用户所有收藏内容
  4. 查看商品收藏状态

售出记录模块

  1. 用户设置商品为售出状态时存入售出记录
  2. 用户分页查看所有的售出记录
  3. 用户删除售出记录

公告模块

  1. 发布公告
  2. 默认展示最新发布的前三条公告
  3. 公告详情的展示
  4. 修改公告及删除公告
  5. 删除公告

后台操作功能

  1. 管理员登录
  2. 查看用户列表
  3. 给予用户封号
  4. 管理员审核商品
  5. 管理员分页展示各类商品信息 全部，已审核:1，待审核:3，违规:0 完成:4
  6. 管理员对商品的操作 违规:0 通过审核:1

网站公告

  1. 管理员发布网站公告

  2. 管理员本人修改或删除公告

  3. 查看网站公告

    通知模块

  4. 评论回复、商品审核、私信发送通知

  5. 设置为管理员或用户发送通知
