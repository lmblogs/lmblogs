windows下的MySQL5.7的下载、安装以及配置环境变量

发表于 2020-02-10 | 分类于 MySQL | 阅读次数:

一、MySQL的安装

1.下载安装包

首先下载安装包到MySQL官网https://www.mysql.com点击downloads

2.点击Community

3.点击MySQL Community Server (GPL)

4.点击MySQL Community Server 5.7

5.选择64位的

注意:这里找不到64位的，但是可以看上面的note其实都是一样的。它会安装64位的文件。

6.点击No thanks, just start my download.

[外链图片转存失败,源站可能有防盗链机制,建议将图片保存下来直接上传(img-2QNrrl2P-1586227898187)(201908061030533b903a93c5a7468fa7cb46887d6f69c5.png)]

7.下载完成双击安装

下面不会具体说了跟着截图做就行

二、下面讲解没有配置MySQL环境变量连接MySQL

1.找到MySQL的安装目录

默认目录为:C:\Program Files\MySQL\MySQL Server 5.7\bin

2.连接数据库

使用mysql -u root -p 回车输入你的密码连接数据库

三、MySQL的环境变量的配置

1.复制刚才的安装目录

首先复制刚才的安装目录C:\Program Files\MySQL\MySQL Server 5.7\bin

2.高级系统设置

对着我的电脑鼠标右键点击属性出现下面的界面点击高级系统设置

3.点击环境变量->双击path

4.点击新建

点击新建，将刚才复制的路径填里面去

5.点击确定退出

点击确定退出，接着打开cmd命令行就可以直接使用命令连接MySQL了
