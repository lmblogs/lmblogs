1、设置root用户(Ubuntu18)

  • 打开文件/etc/pam.d/gdm-autologin 将auth required pam_succeed_if.so user != root quiet_success 注释, 保存。

  • 打开文件/etc/pam.d/gdm-password将 auth required pam_succeed_if.so user != root quiet_success 注释，保存。

  • 打开文件/root/.profile将最后一行修改为：tty -s && mesg n || true
